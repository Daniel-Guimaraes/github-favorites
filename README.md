## 💻 Sobre o projeto

Esta é uma página que permite o usuário salvar os seus usuários favoritos do Github.

## ⚙️ Funcionalidade

- [x] Pesquisa qualquer usuário existente no Github;

- [x] Armazena os usuários pesquisados, no LocalStorage do navegador;
